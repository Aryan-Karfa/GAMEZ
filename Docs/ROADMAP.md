## Version

## Status

# Development Philosophy

GAMEZ will be built using a layered learning approach.

Each phase introduces new concepts while reinforcing previous knowledge.

The goal is not only to finish the application.

The goal is to understand every piece of technology used to build it.

# Final Principle

Do not move to the next phase until the current phase is understood.

The goal is not speed.

The goal is mastery.

A finished project teaches less than a project you truly understand.
